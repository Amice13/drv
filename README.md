# drv

<!-- badges: start -->
<!-- badges: end -->

The goal of drv is to provide data from the Ukrainian State Register of
Voters

## Installation

You can install the released version of drv from
[CRAN](https://CRAN.R-project.org) with:

``` r
install.packages("drv")
```
